# IG DM Only

Check your Instagram DMs without the addictive feed.

A lightweight PWA that redirects you straight to your Instagram DM inbox. No feed, no Reels, no Explore page. Install it on your home screen and use it like an app.

## Install

1. Visit the site in your mobile browser
2. Add to your home screen (iOS: Share → Add to Home Screen / Android: Menu → Add to Home Screen)
3. Open the app — it takes you straight to your DMs

## Privacy

No data collected. No accounts. No tracking. No analytics. Fully open source.

## Development

Serve locally:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
