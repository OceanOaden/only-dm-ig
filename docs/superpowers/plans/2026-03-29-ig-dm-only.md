# IG DM Only Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a static PWA that redirects users to Instagram's DM inbox — and nothing else.

**Architecture:** Seven static files (HTML, CSS, JS, manifest, service worker, 2 icons) with no build step. The single `index.html` serves as both a landing page (in browser) and a redirect launcher (when installed as PWA). Dark mode via `prefers-color-scheme`.

**Tech Stack:** Vanilla HTML, CSS, JS. PWA manifest + service worker. GitHub Pages hosting.

**Spec:** `docs/superpowers/specs/2026-03-29-ig-dm-only-design.md`

---

## File Map

| File | Responsibility |
|------|---------------|
| `index.html` | Landing page structure, meta tags, manifest link, script loading, all page content |
| `style.css` | All styles: layout, typography, light/dark themes, splash screen, responsive |
| `app.js` | Standalone detection, platform detection, install instruction display, redirect logic |

| `manifest.json` | PWA manifest (name, icons, display mode, theme) |
| `icons/icon-192.png` | Placeholder 192x192 icon |

### Task 3: Core JS Logic (detection + redirect + platform instructions)

**Files:**
- Create: `app.js`

- [ ] **Step 1: Create `app.js` with standalone detection and redirect**

```js
(function () {
  'use strict';

  var DM_URL = 'https://www.instagram.com/direct/inbox/';

  // Detect if running as installed PWA
  var isStandalone =
    window.matchMedia('(display-mode: standalone)').matches ||
    window.navigator.standalone === true;

  if (isStandalone) {
    // Show splash and redirect immediately
    document.getElementById('splash').hidden = false;
    window.location.href = DM_URL;
    return;
  }

  // Browser mode — show landing page
  document.getElementById('landing').hidden = false;

  // Platform detection
  var ua = navigator.userAgent;
  var isIOS =
    /iPad|iPhone|iPod/.test(ua) ||
    (navigator.maxTouchPoints > 0 && /Macintosh/.test(ua));
  var isAndroid = /Android/.test(ua);

  // Show platform-specific install instructions
  if (isIOS) {
    document.getElementById('install-ios').hidden = false;
  } else if (isAndroid) {
    document.getElementById('install-android').hidden = false;
  } else {
    document.getElementById('install-desktop').hidden = false;
  }

  // "Show all platforms" toggle
  document.getElementById('show-all-platforms').addEventListener('click', function () {
    document.getElementById('install-ios').hidden = false;
    document.getElementById('install-android').hidden = false;
    document.getElementById('install-desktop').hidden = false;
    this.hidden = true;
  });

  // Register service worker
  if ('serviceWorker' in navigator) {
    navigator.serviceWorker.register('./sw.js');
  }
})();
```
